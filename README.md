# 脚本使用命令
## Shadowsocks (py)
下载脚本 (两种任选其一，或使用其他下载方式)
1. wget
```
wget https://raw.githubusercontent.com/junolym/script/master/shadowsocks/sspy.sh
```
2. curl
```
curl https://raw.githubusercontent.com/junolym/script/master/shadowsocks/sspy.sh > sspy.sh
```
运行脚本
```
chmod +x sspy.sh
sudo bash sspy.sh
```
